# BOB THE FINDER

BOB THE FINDER is a Python script that allows you to find details about a phone number, such as the country, carrier, and line type. It uses the `phonenumbers` library to parse and analyze phone numbers.

## Installation

To be able to use this script, you need to have Python and Git installed on your system. You can download Python from the official website: https://www.python.org/downloads/. You can download Git from the official website: https://git-scm.com/downloads.

Once you have Python and Git installed, you can clone the repository and install the required dependencies. Open a terminal and run the following commands:

```bash
git clone https://github.com/BraveHorizon-gameDev/bob-the-finder.git
cd bob-the-finder
pip install -r requirements.txt
python phonedetail.py
```

## Usage

When you run the script, it will prompt you to enter a phone number. The script will then display the details of the phone number, including the country, carrier, and timezone.
